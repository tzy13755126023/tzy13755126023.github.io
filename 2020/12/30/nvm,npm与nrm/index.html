nvm

nvm（Node Version Manager）是 Nodejs 版本管理器，它能让我们方便的对 Node.js 的版本进行切换。💓

  • Mac 版
  • Windows 版

windows 版安装

  1. 进入项目地址 找到安装包：

  2. 按照引导安装即可。

输入 nvm -v 查看是否安装成功，成功的话如下图所示。如果提示找不到命令，请看这里：传送门

nvm 常用命令

npm

npm（Node Package Manager），Node.js 的包管理器，安装 Node.js 之后自带 npm，无需单独安装。💓

安装失败

输入 npm -v 来查看当前 npm 的版本。


如果出现 nvm 或者 node 安装失败而查找不到命令的情况，请查看这里：传送门

npm 常用命令

nrm

nrm（NPM registry manager），国内使用 npm 官方源来安装包的时候比较慢，所以经常会需要修改 npm 源地址。npm 倒是提供了修改源的方法，但是 nrm 更加方便快捷。💓
