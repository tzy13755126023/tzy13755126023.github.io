前端常用

Vue.js一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。
Vue CLI是Vue.js开发的标准工具。对 Babel、TypeScript、ESLint、PostCSS、PWA、单元测试和 End-to-end 测试提供开箱即用的支持，它的插件系统可以让社区根据常见需求构建和共享可复用的解决方案，通过配套的图形化界面创建、开发和管理你的项目，用单个 Vue 文件即刻实践新的灵感 ，为现代浏览器轻松产出原生的 ES2015 代码，或将你的 Vue 组件构建为原生的 Web Components 组件。
React用于构建用户界面的 JavaScript 库。React 使创建交互式 UI 变得轻而易举。为你应用的每一个状态设计简洁的视图，当数据改变时 React 能有效地更新并正确地渲染组件。
Angular快速构建应用程序并重用您的代码和功能为任何部署目标构建应用程序的方法。对于Web，移动Web，本机移动和本机桌面。
JQuery一个快速，小型且功能丰富的JavaScript库。通过易于使用的API（可在多种浏览器中使用），它使HTML文档的遍历和操作，事件处理，动画和Ajax等事情变得更加简单。兼具多功能性和可扩展性，jQuery改变了数百万人编写JavaScript的方式。
Meteor一组新的技术用于构建高质量的 Web 应用，提供很多现成的包，可直接在浏览器或者云平台中运行。
Aurelia高性能的反应式系统，该系统能够以使其他框架及其虚拟DOM尘土飞扬的方式批量处理DOM更新。无论您的用户界面多么复杂，都可以体验一致，可扩展的性能。
RiotSimple and elegant component-based UI library.
Sassthe most mature
LessIt's CSS
StylusEXPRESSIVE
webpack一个现代 JavaScript 应用程序的静态模块打包器(module bundler)。
GulpA toolkit to automate & enhance your workflow.
GruntThe JavaScript Task Runner.
ECMAScript6The JavaScript Task Runner.
TypeScript An open-source language which builds on JavaScript
微信官方文档微信小程序，微信小游戏，微信公众号，微信支付等等文档入口

UI框架

Ant Design企业级产品设计体系，创造高效愉悦的工作体验。
Element-UI一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库。
Layui经典模块化前端框架，由职业前端倾情打造，面向全层次的前后端开发者，低门槛开箱即用的前端 UI 解决方案。
Bootstrap简洁、直观、强悍的前端开发框架，让web开发更迅速、简单。
View一套基于 Vue.js 的开源 UI 组件库，主要服务于 PC 界面的中后台产品。
AT-UI一款轻量级、模块化的前端 UI 组件库。
BootstrapVue借助BootstrapVue，您可以使用Vue.js和世界上最受欢迎的前端CSS库 Bootstrap v4在网络上构建响应式，移动优先和ARIA可访问的项目 。
Vuetify一个纯手工精心打造的 Material 样式的 Vue UI 组件库。 不需要任何设计技能 — 创建叹为观止的应用程序所需的一切都触手可及。
Vue Material一个轻量级的框架， 建立在谷歌的 Material Design 基础上。 设计强大的和美观的web应用并适用于不同的屏幕。
Buefylightweight UI components for Vue.js based on Bulma.
MUI最接近原生APP体验的高性能前端框架。
Framework7Build full featured iOS
Ant Design Mobile一个基于 Preact / React / React Native 的 UI 组件库。
Cube-UI基于 Vue.js 实现的精致移动端组件库。由滴滴内部组件库精简提炼而来，历经考验，并且每个组件都有充分单元测试，为后续集成提供保障。
Vant 是有赞前端团队开源的移动端组件库，于 2017 年开源，已持续维护 4 年时间。Vant 对内承载了有赞所有核心业务，对外服务十多万开发者，是业界主流的移动端组件库之一。
VUX - Vue移动端UI组件库一个凑合的 Vue.js 移动端 UI 组件库。基于WeUI和Vue(2.x)开发的移动端UI组件库，主要服务于微信页面。
Mand Mobile面向金融场景的Vue移动端UI组件库，丰富、灵活、实用，快速搭建优质的金融类产品，让复杂的金融场景变简单。
FrozenUI一套基于移动端的UI库轻量、精美、遵从手机 QQ 设计规范。
Muse-UI基于 Vue2.0 开发，当下最快的前端框架之一，小巧，api友好，可用于开发的复杂单页应用。自定义主题方式极为优雅，仅需少量代码即可完成主题样式替换。
iView Weapp一套高质量的微信小程序 UI 组件库。
WeUI一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信内网页和微信小程序量身设计，令用户的使用感知更加统一。
VONIC基于 vue.js 和 ionic 样式的 UI 框架，用于快速构建移动端单页应用。
Mint UIVue.js 的移动端组件库。
jQuery WeUI专为微信公众账号开发而设计的一个简洁而强大的UI库，包含全部WeUI官方的CSS组件，并且额外提供了大量的拓展组件，丰富的组件库可以极大减少前端开发时间。

图标库

阿里巴巴矢量图标库阿里妈妈MUX倾力打造的矢量图标管理、交流平台，设计师将图标上传到 iconfont 平台，用户可以自定义下载多种格式的icon，平台也可将图标转换为字体，便于前端工程师自由调整与调用。
Font AwesomeGet vector icons and social logos on your website with Font Awesome
Icons - Material DesignMaterial icons are delightful
Css.ggOpen-source CSS
IoniconsBeautifully crafted open source icons.
Fontello可以根据需求很轻松地制作自定义图标 webfont。.
